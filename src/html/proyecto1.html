Objetivo:

Crear una Página Web Personal como Portafolio y CV Alojada en GitHub

El objetivo de este proyecto es desarrollar una página web profesional que sirva como portafolio personal y CV, y alojarla en GitHub para hacerla accesible públicamente de forma gratuita. Este proyecto tiene como finalidad presentar de manera clara y profesional las habilidades, experiencia y formación de la persona, con el fin de utilizarla como herramienta para la búsqueda de empleo o simplemente para compartir con otras personas. A continuación, se detallan los pasos y consideraciones para lograr este objetivo.

Resolución

  • Definición del Propósito y Contenido de la Página Web
  • Desarrollo de la Página Web
  • Alojamiento en GitHub Pages
  • Optimización y Mantenimiento
  • Beneficios de Usar GitHub Pages

Conclusión

Crear una página web alojada en GitHub para servir como portafolio personal y CV es un proceso relativamente sencillo pero muy poderoso. No solo permite mostrar tu experiencia y habilidades de una forma atractiva y organizada, sino que también demuestra tu conocimiento en desarrollo web y manejo de herramientas modernas como Git y GitHub. Al ser una página pública, es una excelente herramienta para promocionarte de manera profesional, con un impacto visual inmediato que puede ayudarte a destacar en el ámbito laboral o académico.
