Proyecto 1

Este proyecto tiene como objetivo crear una página web que funcione como un portafolio personal y un currículum vitae (CV) en línea. La página debe presentar la información personal, profesional, académica y de contacto de manera organizada y estética. Se utilizarán tecnologías web como HTML, CSS y JavaScript para estructurar, diseñar e interactuar con el contenido. Además, se incluirán funcionalidades interactivas, como un formulario de contacto emergente (popup) y botones con efectos de hover. El proyecto se alojará en GitHub, lo que permitirá un fácil acceso y visualización, así como la implementación de control de versiones.

Proyecto 2

En este proyecto se diseña una red utilizando Cisco Packet Tracer, donde se implementa una configuración de Router on Stick, que permite el enrutamiento de tráfico entre diferentes VLANs en una red. La red consta de dos switches: el Switch1, donde se configura el Router on Stick, y el Switch2, que contiene un servidor DHCP. Este servidor se encarga de asignar dinámicamente direcciones IP a los dispositivos dentro de las VLANs correspondientes. El objetivo es aprender a configurar VLANs, el router para el enrutamiento entre ellas, y un servidor DHCP que automatiza la asignación de direcciones IP, todo con el fin de garantizar una conectividad adecuada en la red.
