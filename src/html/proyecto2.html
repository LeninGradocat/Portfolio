Objetivo:

Red on a stick

El objetivo de este proyecto es diseñar y configurar una red utilizando Cisco Packet Tracer, implementando una topología con un router on stick en Switch1 y un servidor DHCP en Switch2. El diseño de la red incluirá la creación de varias VLANs en ambos switches y la configuración del servidor DHCP para asignar direcciones IP automáticamente a los dispositivos de las VLANs. El propósito de este proyecto es aprender cómo segmentar una red mediante VLANs, cómo configurar un router on stick para permitir la comunicación entre VLANs y cómo implementar un servidor DHCP para gestionar las direcciones IP en una red local.

Resolución

  • Diseño de la Topología
  • Configuración de VLANs
  • Verificación de la Conectividad
  • Configuración del Router on Stick
  • Configuración del Servidor DHCP

Conclusión

Este proyecto permitió implementar una red segmentada utilizando VLANs para mejorar la seguridad y eficiencia del tráfico de red. La configuración del router on stick facilitó la comunicación entre las VLANs, mientras que el servidor DHCP simplificó la gestión de direcciones IP al automatizar su asignación. Al final, se logró crear una red funcional y organizada en Cisco Packet Tracer, cumpliendo con los objetivos de segmentación de red, enrutamiento inter-VLAN y asignación automática de direcciones IP mediante DHCP. Este proyecto refuerza conceptos fundamentales sobre VLANs, enrutamiento y administración de direcciones IP en redes locales.
